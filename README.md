CcImage
=======

一个使用opencv和MFC做的数字图像处理小应用


功能
======

Color：
------------

* binary
* otsu
* invert
* gray


Morph：
------------

* Erode
* Dilate
* Closing
* Opening
* Gardient
* Grabcut
* Wateres



Filter：
------------

* Average
* GuassianBulr
* MeduimiamBlur


Edge：
------------

* Canny
* Sobel
* Laplace
* Corerns
